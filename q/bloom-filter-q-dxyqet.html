实现一个简单的布隆过滤器（Bloom Filter）

中等

核心思想

布隆过滤器用一个位数组和多个哈希函数表示集合。插入元素时，把多个哈希位置都置为 1；查询时只要有一个位置为 0，就一定不存在；全部为 1 则表示“可能存在”。

type Bloom struct {
    bits []bool
}

func (b *Bloom) Add(s string) {
    for _, idx := range []int{hash1(s), hash2(s), hash3(s)} {
        b.bits[idx%len(b.bits)] = true
    }
}

func (b *Bloom) MightContain(s string) bool {
    for _, idx := range []int{hash1(s), hash2(s), hash3(s)} {
        if !b.bits[idx%len(b.bits)] {
            return false
        }
    }
    return true
}

特点

  • 优点：空间效率高，查询快
  • 缺点：有误判，不支持精准删除
  • 常见用途：缓存穿透防护、URL 去重、黑名单预判
