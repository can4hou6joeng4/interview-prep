# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/) spirit:

- Be respectful and constructive
- Assume good intent
- Focus feedback on ideas and implementation, not people
- Help keep the repository welcoming for learners and contributors

Unacceptable behavior includes harassment, personal attacks, discriminatory language, or repeated disruptive conduct.

For moderation or conduct concerns, please contact the repository maintainer privately through GitHub.
