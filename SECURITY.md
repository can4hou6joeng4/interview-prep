# Security Policy

## Supported Scope

当前仓库是静态 GitHub Pages 项目，没有后端服务或数据库。

仍然欢迎反馈以下安全相关问题：

- GitHub Actions 配置泄露风险
- GitHub Pages 发布产物包含不应公开的文件
- 浏览器端脚本存在明显注入风险
- 本地持久化数据处理存在意外暴露问题

## Reporting

- 普通内容错误、题目错漏、样式异常：请直接提交 Issue。
- 涉及安全或敏感信息的问题：不要公开提交 Issue，请通过 GitHub 私下联系仓库维护者。

## Response Expectations

- 会优先确认是否能复现。
- 确认问题后会尽快修复并更新公开说明。
