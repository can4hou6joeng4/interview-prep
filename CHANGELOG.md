# Changelog

本文件记录仓库中对用户可见、对仓库质量有明显影响的更新。

## 2026-04-20

### Added

- 原生 iOS App 新增独立的复习完成页 `ReviewCompletionView`，展示掌握率环形图、专题分布卡片以及最弱专题回跳按钮。
- 深链新增 `.category(slug:)` 目标，支持从复习完成页直接跳到指定分类并推入 NavigationStack。
- 搜索结果行新增右侧 Menu 快捷动作，可直接收藏、加入学习中，或跳进所属专题。
- 分类题目页接入掌握度进度条与状态筛选（未学 / 学习中 / 已掌握 / 收藏），行内加上状态角标。
- 首页新增连续学习、本周浏览、本月专题三项激励指标卡，沉淀学习节奏。
- 题目详情页新增一键「加入今日复习」开关，支持在未答题状态直接标记进队列。
- 我的页笔记模块接入关键词检索，可同时匹配笔记正文、题目和所属分类。
- 设置页新增学习数据区块，支持 JSON 导出和本地学习进度的二次确认重置。

### Changed

- 复习会话在首轮完成后不再停留在单卡片总结，而是进入结构化的完成态，给出下一步建议。
- README 中英文版同步 iOS 新能力并勾选 `2026-04-19-ios-native-study-tool.md` 内的 Task 1-5，并追加 2026-04-20 增量条目。

## 2026-04-17

### Added

- 新增题目独立页（`q/*.html`，共 203 页）与分类聚合页（`c/*.html`，共 19 页），全部静态生成，便于搜索引擎独立索引长尾流量。
- 新增 `scripts/slug.mjs`、`scripts/build-pages.mjs`、`scripts/validate-pages.mjs` 三个零依赖脚本，覆盖 slug 生成、页面构建、生成物校验三个环节。
- `sitemap.xml` 从 3 条 URL 扩充到 225 条，覆盖首页、学习页、模拟面试页、19 个分类页和 203 个题目页。

### Changed

- `assets/data.js` 每个分类新增 slug 字段；`scripts/validate-data.mjs` 新增 slug 必填、格式与唯一性校验。
- `scripts/check-fast.sh` 追加 slug 单测与生成物校验，提交前自动拦截回归。
- GitHub Actions 在现有校验之后追加生成物同步性检查，会拒绝「改了 data.js 但没重新 build」的提交。
- `assets/styles.css` 复用既有 Neo-Brutalism token 补齐独立页所需的面包屑、难度徽章、答案区、相关题和主行动按钮样式。

## 2026-04-08

### Added

- 新增 3 道正式题，补齐分布式存储、Go 网络框架选型和性能优化方法论。
- 新增专项复习入口，支持在当前学习路径内按错题、模糊题、薄弱题和已掌握题快速切换复习范围。
- 新增学习路径专题入口，将现有题库编成跨境交易、平台治理、实时系统、项目深挖和面试冲刺 5 条训练路径。
- 新增按标签过滤，支持在卡片、列表和模拟面试模式中按"简历项目 / 场景题"快速缩小题目范围。
- 新增 12 道正式题，覆盖资源管理系统建模、配置中心分层覆盖、协议选型、容量评估、金额分摊与尾差、订单快照、语音房/直播间模块、程序化广告统计链路、自动化工单系统、时序数据库边界、压测方案、告警到回滚闭环。
- 新增 5 道 AI Agent 相关正式题，覆盖 Agent 框架选型、失败场景治理、规划方法取舍、企业客服 Agent 架构。
- 新增 8 道高频面试题，覆盖 WaitGroup、select 调度、协程池、sync.Map 引用安全、Redis 宕机降级、MQ 选型、gRPC 底层机制。
- 新增 `模拟面试` 模式，支持掌握回顾、冲刺提升和真实混合三种题源。
- 模拟结果页新增薄弱题再练一轮功能。
- 站点新增最近更新面板。

### Improved

- 切换学习路径时自动回到干净的默认筛选状态。
- 模拟结果页补充薄弱分类统计和下一轮训练建议。
